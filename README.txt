*********Hướng dẫn thực thi chương trình*********

1 Hướng dẫn cài đặt 
Bước 1: Tải và cài đặt Nodejs (https://nodejs.org/en/download)
Bước 2: Mở CMD tại thư mục lưu trữ source code
Bước 3: Gõ lệnh “npm install” (thư mục node modules sẽ được khởi tạo)

2 Hướng dẫn sử dụng
Cách 1: Truy cập vào đường link: 
Cách 2: Trong trường hợp link trang web gặp trục trặc thì có thể thực thi theo cách dưới đây. 
	Bước 1: Sử dụng Visual Studio để khởi chạy Server phía Back-end
	Bước 2: Gõ lệnh “npm start” trong CMD để khởi chạy Server phía Front-end
	Bước 3: truy cập vào đường link http://localhost:3000 để truy cập vào website
